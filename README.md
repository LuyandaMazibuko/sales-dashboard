# 📊 Sales Dashboard Analysis (Python Project)

## 📌 Overview
This project analyzes sales data to generate business insights using Python.

The goal is to identify:
- Best and worst performing months
- Top-performing products
- Revenue distribution by category

---

## 🛠️ Technologies Used
- Python
- Pandas
- Matplotlib

---

## 📂 Dataset
The dataset contains:
- Product names
- Categories
- Sales revenue
- Dates

---

## 📈 Key Features
- Data cleaning and preprocessing
- Revenue analysis by:
- Product
- Category
- Month
- Automated insights generation
- Data visualization (charts)

---

## 📊 Output Examples
- Monthly revenue trends
- Product performance comparison
- Category revenue breakdown

---

## 💡 Insights
- Identified top-performing product: **Wireless Charger**
- Best month: **March**
- Worst month: **April**
